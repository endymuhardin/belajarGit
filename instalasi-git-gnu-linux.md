# Instalasi Git di GNU/Linux
Meski di berberapa distro GNU/Linux telah di sediakan git secara bawaan,akan tetapi tidak semua distro membawa git secara bawaan di dalam distronya
## Arch Linux
```
pacman -S git
```
## Debian
```
apt install git
```
## Slackware
```
slackpkg install git
```
## Centos/Fedora/RHEL
```
yum install git
```
