Installasi git di windows

1. Pertama download git di https://git-scm.com/downloads, pilih version untuk windows.

2. Kemudian install git seperti biasa, seperti menginstall aplikasi atau program pada umumnya. Tinggal next saja hingga muncul proses instalasi.

3. Proses installasi git sedang berlangsung, mohon untuk menunggu beberapa menit hingga proses selesai.

4. Proses installasi git telah selesai dan berhasil.

5. Apabila proses intallasi git telah berhasil, maka akan muncul 3 shortcut yaitu Git Bash, Git CMD, Git GUI
